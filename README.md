## 📖 About This Project

This is a from‑scratch implementation of a physically‑based ray tracer, written in C++17.

The goal is to build a complete, production‑style renderer while learning:
- **Graphics Foundations**: Ray‑object intersection, shading, materials, and camera models.
- **Mathematical Tools**: Vector algebra (dot/cross products), linear transformations, and Monte Carlo integration.
- **C++ Modern Idioms**: RAII, move semantics, `constexpr`, smart pointers, and template metaprogramming.
- **Performance Engineering**: Cache‑aware data layouts, SIMD‑friendly code, and multi‑threading.
- **Software Architecture**: Separation of concerns, dependency injection, and test‑driven development (with unit tests)
